    组件介绍

    # 组件介绍

    商品详情介绍页。包括商品详情展示、图片放大展示、加入购物车操作等。

    # 进入离开组件

    1、由Search路由组件，路由跳转进入；参数：商品id

    2、由AddCartSuccess路由组件，路由跳转进入；参数：商品id

    3、跳转进入AddCartSuccess路由组件；参数：商品信息及加入购物车数量。

    # 子组件

    Zoom商品图片放大镜组件、ImgList商品图片轮播图组件

    # 生命周期

    1、mounted () 调用vuex-actions-getGoodInfo：向服务器发送”获取商品详情数据“的网络请求，如果服务器返回的result正常，则将返回的data保存到state中，以供组件使用。

    # 数据

    名称 数据类型 所属 数据介绍
    skuNum 数值 data 商品加入购物车数量
    categoryView 对象 computed 组件从仓库state中获取的商品所处分类数据
    skuInfo 对象 computed 组件从仓库state中获取的商品详情数据
    spuSaleAttrList 数组 computed 组件从仓库state中获取的商品加入购物车的属性数据
    skuImageList 数组 computed 从skuInfo中分解出来的图片轮播图数据

    # API

    1、获取商品详情数据

    地址：/api/item/{skuId}   请求方式：get    需要带参数skuId
export const reqGetGoodInfo = (skuId) =>
  requests({ url: `/item/${skuId}`, method: "get" });

    2、商品加入购物车

    将商品加入购物车   地址：/api/cart/addToCart/{skuId}/ {skuNum}  请求方式：post    需要带参数skuId, skuNum
export const reqAddOrUpdateCart = (skuId, skuNum) =>
  requests({ url: `/cart/addToCart/${skuId}/${skuNum}`, method: "post" });

    # vuex

    vuex-detail：
1、actions
1.1、getGoodInfo
调用API-获取商品详情数据；服务器返回请求成功，则调用mutations-GETGOODINFO
1.2、addOrUpdateCart
调用API-商品加入购物车；服务器返回请求成功，则通知组件成功

2、mutations
2.1、GETGOODINFO
将服务器返回的商品详情数据保存到state的goodInfo中，以供其他组件调用

3、state
goodInfo: {}, 			// 商品详情数据，对象

4、getters
categoryView			// 商品详情中的商品所处分类数据，对象	
skuInfo					// 商品详情中的商品属性数据，对象	
spuSaleAttrList			// 商品详情中的商品加入购物车的属性数据，数组

    # methods

    1、addCart() 加入购物车

    1、参数：无

2、函数内容：
2.1、dispatch触发addOrUpdateCart；
2.2、请求返回成功，则将商品属性数据存入sessionStorage，以供其他组件查看；
2.3、跳转向加入购物车成功路由组件

    # 功能点

    1、页面数据动态展示

    组件在mounted阶段，调用vuex的getGoodInfo；vuex调用获取商品详情数据的API，向服务器发送请求；服务器返回成功则将返回的data保存到state中；本组件的computed获取vuex-state数据，并保存为计算属性；页面在展示时，使用相应的属性即可。

    2、加入购物车

    用户点击加入购物车按钮，组件将商品id及数量作为参数，调用vuex的addOrUpdateCart；vuex调用获取加入购物车的API，向服务器发送请求；服务器返回成功则将其他组件需要的数据会话存储，路由跳转至加入购物车成功组件。

    # 子组件ImgList

    # 进入组件

    作为Detail路由组件的子组件固定展示；参数：父组件传递的数据skuImageList

    # 数据

    名称 数据类型 所属 数据介绍
    currendIndex number data 要展示的图片索引值
    skuImageList 数组 props 父组件传递过来的商品图片轮播图数据

    # methods

    1、changeCurrentIndex(index) 用户点击商品图片列表中的某一张图片

    1、参数：index 用户点击的图片的索引值

2、函数内容：
2.1、将索引值保存到data-currendIndex中
2.2、触发全局自定义事件getIndex，将索引值传递给兄弟组件Zoom

    # watch

    1、$skuImageList 监听图片轮播图数据的变化，以实现轮播图切换

    # 功能点

    1、页面数据动态展示

    获取父组件传递的轮播图数据，结合Swiper插件进行轮播图展示。

    2、用户点击商品图片列表中的某一张图片

    保存用户点击图片的索引值，并传递给兄弟组件Zoom。

    # 子组件Zoom

    # 进入组件

    作为Detail路由组件的子组件固定展示；参数：父组件传递的数据skuImageList

    # 生命周期

    1、mounted () 添加全局自定义事件getIndex，获取兄弟组件ImgList中用户选中的图片索引值，赋值给currendIndex。

    # 数据

    名称 数据类型 所属 数据介绍
    currendIndex number data 要展示的图片索引值
    skuImageList 数组 props 父组件传递过来的商品图片轮播图数据
    imgObj 对象 computed 从skuImageList中分解出来的用户选中的要展示图片数据

    # methods

    1、handler() 鼠标移动，展示商品图片的放大区域。

    # 功能点

    1、页面数据动态展示

    添加全局自定义事件getIndex，获取兄弟组件ImgList中用户选中的图片索引值。根据索引值，从商品图片轮播图中找到图片进行展示；

    2、鼠标进入图片放大展示

    用户鼠标进入商品展示图，在右侧浮现展示图的放大版图片，并随着用户鼠标同步移动。

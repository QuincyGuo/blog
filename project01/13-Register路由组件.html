    组件介绍

    # 组件介绍

    用户注册。

    # 进入离开组件

    1、由Header组件，路由跳转进入；参数：无

    2、跳转进入Login路由组件；参数：无

    # 数据

    名称 数据类型 所属 数据介绍
    phone string data 从页面表单上获取的用户输入的手机号
    code string data 用户输入的验证码
    password string data 用户输入的密码
    password2 string data 用户输入的确认密码
    agree boolean data 用户是否勾选同意协议
    phoneError string computed phone不符合格式存在的问题
    passwordError string computed password不符合格式存在的问题

    # API

    1、获取注册验证码

    地址：/api/user/passport/sendCode/{phone}  请求方式：get   需要带参数phone
export const reqGetCode = (phone) =>
  requests({ url: `/cart/user/passport/sendCode/${phone}`, method: "get" });
// 服务器收到请求后，会将验证码作为data返回浏览器

    2、用户注册

    地址：/api/user/passport/register  请求方式：post   需要带参数data{ phone, password, code }
export const reqUserRegister = (data) =>
  requests({ url: "/user/passport/register/", data, method: "post" });

    # vuex

    vuex-user：
1、actions
1.1、getCode
调用API-获取注册验证码；服务器返回请求成功，则调用mutations-GETCODE
1.2、userRegister
调用API-用户注册；服务器返回请求成功，则通知组件成功

2、mutations
2.1、GETCODE
将服务器返回的验证码保存到state的testCodeFromServer中，以供其他组件调用

3、state
testCodeFromServer:'' 			// 服务器返回的验证码，字符串

    # methods

    1、getCode() 获取注册验证码

    1、参数：无

2、函数内容：

2.1、dispatch触发getCode；
2.3、请求返回成功，组件获取vuex-state-testCodeFromServer，并将其填入页面上的验证码输入框；

    2、userRegister() 用户注册

    1、参数：无

2、函数内容：
2.2、dispatch触发userRegister；
2.3、请求返回成功，路由跳转至Login路由组件。

    # 功能点：

    1、获取注册验证码

    用户点击按钮，调用vuex的getCode；vuex调用获取注册验证码的API，向服务器发送请求；服务器返回成功则将返回的data保存到state中；本组件的data获取vuex-state数据；页面在展示时，使用相应的属性即可。

    2、用户注册 用户点击按钮，调用vuex的userRegister；vuex调用用户注册的API，向服务器发送请求；组件获取服务器的请求返回结果，如果成功，组件跳转至登录路由组件

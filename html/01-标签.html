    自闭和标签

    # 自闭和标签

    一般标签都是需要一头一尾，开始和结束标签，意味着可以在标签内部插入其他标签或文字；

    但是有些特殊标签，只有开始标签，没有结束标签，因此不可以在标签内部插入其他标签或文本，只能在内部写入标签的属性。

    <!-- 一般标签 -->
<h1>hahaha</h1>

<!-- 自闭和标签 -->
<img src = "testLink" alt = "photoIntroduction">
<input type = "text" id = "testid" name = "testname">
<br>换行
<hr>水平线

    # 文本&容器级标签

    <!-- 文本标签 -->
<p>、<span>、<a>、<b>、<i>、<u>
<!-- 文本级标签内只能嵌套文字、图片，无法再嵌套其他标签 -->

    <!-- 容器标签 -->
<div>、<h>、<ul>、<li>、<dl>、<dt>、<tr>、<td>
<!-- 容器级标签内能嵌套其他标签 -->

    HTML的文本级、容器级标签，对应的是CSS的行内元素、块级元素。

    # 特殊标签

    浏览器呈现HTML时，会将程序员编写文本时输入的多个空格、制表符，回车，换行，都压缩成一个空格。

    # 回车与换行

    '\r' 回车，回到当前行的行首，而不会换到下一行，如果接着输出的话，本行以前的内容会被逐一覆盖；
'\n' 换行，换到当前位置的下一行，而不会回到行首；
所以，Windows系统里，每行结尾是回车换行\r\n。但是Mac系统和Unix系统则不一样。

但是，HTML有自己的换行标签，是<br>。

    空格	&nbsp
换行	<
<		&lt     
>   	&gt

    # 上标下标

    上标	<sup>
下标	<sub>
<P>
    3<sup>2</sup>,5<sub>3</sub>
</P>
显示为：3(下标2), 5(上标3)

    /** 在所有浏览器中防止 sub 和 sup 影响 line-height **/
sub, sup { 
  font-size: 75%; 
  line-height: 0;
  position: relative; 
  vertical-align: baseline; 
} 
sup { 
  top: -0.5em; 
} 
sub { 
  bottom: -0.25em; 
}

    # 元素焦点 tabindex

    <p tabindex="0">Tabbable due to tabindex.</p>
<!-- 
指示其元素是否可以聚焦，以及它是否/在何处参与顺序键盘导航。比如上面代码就是让一段文字可以获得焦点。

-  tabindex = "-1"，元素可聚焦，但不能通过键盘导航访问到，用JS做页面小组件内部键盘导航的时候非常有用。
-  tabindex = 正值 ，元素可聚焦，可以通过键盘导航访问到；相对顺序按照tabindex的数值递增而滞后获焦。如果多个元素拥有相同的 tabindex，相对顺序按照在DOM中的先后顺序决定。
-  tabindex = "0"或自带可获得焦点属性的元素比如 li ，表示元素是可聚焦的，可以通过键盘导航访问到该元素；此类元素顺序在在tabindex为正值的元素后面，然后按照DOM先后顺序决定。
-->

    # 超链接

    URL：统一资源定位符。也叫地址，路径。互联网上的每个文件都有唯一的URL。

    http://www.site.com/web/index.html

http	模式<!-- 模式告诉浏览器如何处理需要打开的文件。http（Hypertext Transfer Protocol，超文本传输协议）、https -->
www.site.com	主机名
web/index.html	文件路径
web	目录
index.html	文件名

    # 超链接标签介绍

    <a href="链接目标地址" target="_blank" rel="external" title="Learn more at runoob">链接显示内容</a>
rel属性说明，当前文档与被链接文档之间的关系；
target属性，规定在何处打开目标URL；
title属性是，链接提示，鼠标划过链接时显示的文字；文字应该是对链接显示内容的提示，比如，不用“点击此处”，而是“了解销售情况”。

    # 链接跳转到网页的某一部分，锚链接

    比如返回顶部链接按钮。

    标记部分： 
<a id="top">顶部位置</a> 
链接调用部分： 
<a href = "#top">返回顶部</a> 
<a href = "xxx.html#top">返回其他网页的标记位置</a>

    # 图片链接

    链接显示内容通常是文字，但也可以是图片或者其他HTML元素。

    <a href="www.baidu.com">     
    <img border="0" src="/i/eg_buttonnext.gif" /> 
</a>

    # 块级链接

    链接内容不仅是图片、文字的时候：

    <article>   
    <a href="story1.html">     
        <h3>Bruce Lawson voted sexiest man on Earth</h3>     
        <p><img src="bruce.jpg" alt="gorgeous lovebundle. ">A congress representing all the planet's women unanimously voted Bruce Lawson as sexiest man alive.</p>     
        <p>Read more</p>   
    </a> 
</article>

    # 绝对路径与相对路径

    # 相对URL

    以文件本身的位置为参照点，要求与被引用文件要在同一个服务器/主机上。

    # 常用相对路径

    1、
../表示上一层路径，      	<img src = “../images/22.jpg”> 
同一目录下的文件，			<img src = “22.jpg”> <img src = “./22.jpg”>	
同一目录下的文件夹里的文件，  <img src = “images/22.jpg”>

2、@：src文件夹
js里，文件路径，@代表src文件夹；CSS里，~@代表src文件夹
import { 。。。 } from "@/utils/token";	// 不需要从本级文件一路../../往上寻找，只要开头@/就代表已定位在src文件夹下
如果@代表src失败，则在项目的jsconfig.json中设置，
"compilerOptions": {
    "paths": {
      "@/*": ["src/*"]
    },
}

3、root：根目录
假如需要向上翻很多层才能找到，则不使用“../../../images/22.jpg”，太繁琐，而是可以直接从根目录开始向下
<img src = “/root/styles/tablestyle.css”>

    # 绝对URL

    包含了指向文件的路径的完整信息，从http到主机名到文件名。

    # head标签

    这一部分的内容包括页面信息、内外部文件链接、移动端适配等内容。

    <!DOCTYPE html>	
<!-- 告诉浏览器这是一个HTML页面，始终位于第一行 -->

<html lang="en">
<!-- 网页语言，中文统一用"zh-cn" -->

    # 网页字符集

    <meta charset="UTF-8">
<!-- 
一般是以下两种选择：
UTF-8	：各国字符齐全，体积大；  
gb2312	：只有中文字符，体积小； 
-->

    # 页面作者和页面描述

    <meta name="author" content="Chris Mills">
<!-- 页面作者：能让一些管理系统获取到页面作者信息 -->
<meta name="description" content="The MDN Learning Area aims to provide complete beginners to the Web with all they need to know to get started with developing web sites and applications.">
<!-- 页面描述：能在搜索引擎中获得更高的相关性排名 -->

    # 页面标题

    <title>为什么加入其他国家国籍就意味着不爱国？ - 知乎 </title>
<!-- 对搜索引擎中的排名最重要。选择最能简要概括网页内容的关键字。网页特有文字-网站名称 -->

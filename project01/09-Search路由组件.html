    组件介绍

    # 组件介绍

    商品搜索结果的展示页，包括商品结果分页展示、对商品结果进行细分搜索等。

    # 进入离开组件

    1、由TypeNav组件，路由跳转进入；参数：category1Id 一级分类导航被选择的标签id，category2Id 二级分类的id，category3Id 三级分类的id，categoryName 分类导航选择的分类名

    2、由Header组件，路由跳转进入；参数：搜索关键字keyword。

    3、跳转进入Detail路由组件；参数：商品id。

    # 子组件

    SearchSelector搜索条件筛选组件、Pagination分页器组件、TypeNav三级导航分类静态组件

    # 生命周期

    1、mounted ()

    1.1、保存路由跳转进入时携带的参数，保存在data中。

    1.2、调用getData方法，调用vuex-actions-getSearchList：向服务器发送”搜索商品结果数据“的网络请求，如果服务器返回的result正常，则将返回的data保存到state中，以供组件使用。

    # 数据

    名称 数据类型 所属 数据介绍
    searchParams 对象 data 搜索条件：由其他组件路由跳转进入时携带的参数/搜索子选择组件的筛选条件/分页器参数
    goodsList 数组 computed 组件从仓库state中获取的商品搜索结果数据
    orderLabelOne/orderLabelTwo 布尔值 computed 综合/销量排序标签是否被选中
    orderLabelAsc/orderLabelDesc 布尔值 computed 升/降序排序标签是否被选中
    total 数值 computed 组件从仓库state中获取的商品搜索结果总数

    1、获取搜索结果商品数据

    地址：/api/list   请求方式：post    需要带参数data

    # 功能点

    1、页面数据动态展示

    组件在mounted阶段，保存路由跳转进入时携带的参数，保存在data中；调用vuex的getSearchList；vuex调用获取搜索商品结果数据的API，向服务器发送请求；服务器返回成功则将返回的data保存到state中；本组件的computed获取vuex-state数据，并保存为计算属性；页面在展示时，使用相应的属性即可。

    2、搜索关键字、搜索TypeNav分类导航

    用户搜索关键字/点击商品分类导航内某一分类，搜索框及分类导航所在的组件将关键字及选中的分类属性加入到跳转要携带的参数中；更新路由url；获取路由路径刷新时携带的新参数，更新data；然后向服务器获取搜索结果商品数据，刷新页面。

    3、SearchSelector子组件的商品筛选

    用户点击选择SearchSelector子组件的商品筛选条件，更新data；然后向服务器获取搜索结果商品数据，刷新页面。

    4、去除面包屑导航的搜索标签

    用户点击去除面包屑导航的搜索标签，更新data；然后向服务器获取搜索结果商品数据，刷新页面。

    5、更换页码

    用户点击更换页码，更新data；然后向服务器获取搜索结果商品数据，刷新页面。

    6、更换商品结果排序

    用户点击商品结果排序，更新data；然后向服务器获取搜索结果商品数据，刷新页面。

    # 子组件Pagination

    # 进入组件

    作为Search路由组件的子组件固定展示；参数：父组件Search添加的自定义事件getPageNo，父组件Search传递的数据pageNo、pageSize、total、continues。

    # 数据

    名称 数据类型 所属 数据介绍
    pageNo 数值 props 目前所在第几页
    pageSize 数值 props 每页展示的数据个数
    total 数值 props 总共要展示的数据个数
    continues 数值 props 页码连续展示多少页
    totalPage 数值 computed 总共要展示的页数
    startNumAndEndNum 对象 computed 连续页码中的起始与结束页位置

    # methods

    1、prePageClick() 点击上一页

    1、参数：无

2、函数内容：
2.1、触发父组件Search绑定的自定义事件getPageNo，向其传递pageNo - 1

    2、pageOneClick() 点击第一页

    1、参数：无

2、函数内容：
2.1、触发父组件Search绑定的自定义事件getPageNo，向其传递pageNo:1

    3、pageClick(index) 点击中间的连续页

    1、参数：index，点击的页码

2、函数内容：
2.1、触发父组件Search绑定的自定义事件getPageNo，向其传递pageNo:index

    4、pageEndClick() 点击最后一页

    1、参数：无

2、函数内容：
2.1、触发父组件Search绑定的自定义事件getPageNo，向其传递pageNo:totalPage

    3、nextPageClick() 点击下一页

    1、参数：无

2、函数内容：
2.1、触发父组件Search绑定的自定义事件getPageNo，向其传递pageNo + 1

    # 功能点

    1、分页器数据动态展示

    组件的props获取父组件传递过来的pageNo、pageSize、total、continues数据；computed计算出totalPage总共页数、startNumAndEndNum连续页码中的起始与结束位置；展示分页器时，使用相应的属性即可。

    2、点击上一页/下一页等按钮

    触发父组件Search绑定的自定义事件getPageNo，向其传递用户点击后的页码。

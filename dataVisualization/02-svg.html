    介绍

    # 介绍

    SVG是一种基于XML语法的图像文件格式，缩放不失真的矢量图形。

    # 初始化画布

    <svg class="box" viewBox='0 0 800 300'></svg>
.box {
  width: 800px;
  height: 300px;
  border: 1px solid rosybrown;
}
// <viewBox>属性的值有四个数字，分别是画布左上角的横坐标和纵坐标、视口的宽度和高度。

    # 在画布中绘制图形

    # 样式

    fill：设置填充颜色		fill-opacity：填充色的透明度		stroke：线的颜色		stroke-width：线宽

    # 线段

    <line x1="10" y1="20" x2="150" y2="20" stroke="red" stroke-width="2px"></line>
// 需要起点、终点的xy坐标

    # 折线

    <polyline points="10 50, 10 100, 120 50, 120, 100" fill-opacity="0" stroke="blue"></polyline>
// 需要起点、折线中途拐点、终点的xy坐标

    # 矩形

    <rect x="200" y="30" width="150" height="50" fill="pink"></rect>
// 需要矩形左上角点的xy坐标，矩形的宽高

    # 圆

    <circle cx='450' cy='60' r='50' style='stroke: cyan; fill:none'></circle>
// 需要圆心的xy坐标，圆半径

    # 椭圆

    <ellipse cx="650" cy="60" rx="80" ry="30" style="fill: black;"></ellipse>
// 需要圆心的xy坐标，椭圆横轴、纵轴的半径

    # 多边形

    <polygon points="30 150, 150 150, 70 200" stroke="red" fill-opacity="0" />
// 需要多边形每个端点的xy坐标

    # 路径

    点与点之间连线，绘制任意图形

    <path d="M 250 150 L 300 200 L 350 150 L 400 200 L 450 150 L 350 120 Z"></path>
// 字母表示绘制动作：M 设置起点 L连线 Z 终点与起点闭合；字母后面的两个数字是点的坐标

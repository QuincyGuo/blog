    函数也是对象的一种

    # 函数也是对象的一种

    函数也属于对象，对象类型是function。

    # 函数的定义/声明

    # 方式一：function关键字

    function fun1(a,b)
{
  return a+b;
}

    # 方式二：函数表达式

    var 变量名 = function(a,b)
{
  语句....
}
所谓的“函数表达式”，其实就是将匿名函数（无名函数）赋给一个变量。这个变量里保存的是一个函数，而不是一个值。

    其实，函数是一个function对象，他是保存在堆内存中的，而函数名是一个变量，保存在栈内存中。所以，这两种方式的表达方式并没有本质上的区别，函数名和这个函数表达式（变量）都是名称和内容分别保存。

    # 函数的调用

    # 方式1：普通函数的调用

    代码举例：
function fn1() {
	console.log('我是函数体里面的内容1');
}
fn1(); 		// 调用函数
fn1.call(); // 调用函数

    # 方式2：通过对象的方法来调用

    var obj = {
	a: 'qianguyihao',
	fn2: function() {
		console.log('千古壹号，永不止步!');
	},
};
obj.fn2(); // 调用函数

    # 方式3：立即执行函数

    (function() {
	console.log('我是立即执行函数');	//立即执行函数在定义后，会自动调用。
})();
立即执行函数往往只会执行一次。为什么呢？因为没有变量保存它，执行完了之后，就找不到它了。

    # 参数相关

    - 函数的实参可以是任意的数据类型。所以要注意，是否有可能会接收到非法的参数，如果有可能则需要对参数进行类型的检查。
- 如果实参的数量少于形参的数量，多余的形参会被定义为 undefined。表达式的运行结果为 NaN。

    # return 相关

    - 如果return语句后不跟任何值，就相当于返回一个undefined
- 如果函数中不写return，则也会返回undefined
- 返回值可以是任意的数据类型，可以是对象，也可以是函数，也可以是一个表达式（return a<b; 返回的是布尔值）。
- return 只能返回一个值。如果用逗号隔开多个值，则以最后一个为准。

    # fn() 和 fn 的区别

    • fn()：调用函数。调用之后，还获取了函数的返回值。
    • fn：函数对象。相当于直接获取了整个函数对象。
    function name2(a,b)
{
	return a + b;
}
console.log(name2);
console.log(name2(a,b));
上面两句代码执行的结果是不同的。第一句结果是，将整个函数体显示出来；第二句是显示函数返回结果。

    # 函数作为返回值

    函数作为返回值，一般用于回调函数

    以函数防抖代码为例：
function debounce(fn, delay) {
    var timer; // 定义一个timer
    return function () {	// 这里是将函数作为返回值
      clearTimeout(timer); 
      timer = setTimeout(function () {fn()}, delay); 
    };
}
函数调用：
input.oninput = debounce(function () {
    console.log("ajax发请求");
}, 1000);
// 执行完后，input.oninput的内容会是一个函数，也就是，每次oninput都会执行这段函数代码。

但如果，写成子函数定义使用，等于只是调用了一次debounce，给input.oninput没留下什么，并不意味着每次触发input.oninput就调用debounce函数
function debounce(fn, delay) {
    var timer; // 定义一个timer
    function aa() {	// 这里是将函数作为返回值
      clearTimeout(timer); 
      timer = setTimeout(function () {fn()}, delay); 
    };
    aa();
}

    静态组件

    # 静态组件

    # 组件介绍

    实现对某条spu类添加一个sku实例的新增操作。

    # 进入离开组件

    作为Spu组件的子组件，由用户在Spu组件，点击“添加SkU”按钮，进入本组件。

    父Spu组件为本组件绑定了一个自定义事件changeScene2，以确定从本组件返回时页面的显示区域。

    # 数据

    名称 数据类型 所属 数据介绍
    spu Object data Spu组件传过来的当前sku对应的spu信息
    attrInfoList Array data 该SPU所属的三级分类的平台属性数据；sku的平台属性从中选择
    spuSaleAttrList Array data 该SPU下所有的销售属性数据；sku的销售属性从中选择
    spuImageList Array data 该SPU下所有的商品图片数据；sku的图片属性从中选择
    skuInfo Object data 与添加sku页面里的表单绑定的数据
    selectedImgList Array data 用户从图片列表中，选中的图片数据

    # API

    1、获取某条spu类的全部图片

    // 路径：/admin/product/spuImageList/{spuId}	请求方式：get	参数：spuId
export const reqSpuImageList = (spuId) =>
  request({ url: `/admin/product/spuImageList/${spuId}`, method: "get" });

    2、获取某条spu类的全部销售属性数据

    // 路径：/admin/product/spuSaleAttrList/{spuId}	请求方式：get	参数：spuId
export const reqSpuSaleAttrList = (spuId) =>
  request({ url: `/admin/product/spuSaleAttrList/${spuId}`, method: "get" });

    3、获取某个三级商品分类下的全部平台属性数据

    // 路径：/admin/product/attrInfoList/{category1Id}/{category2Id}/{category3Id}		请求方式：get	参数：category1Id		category2Id		catrgory3Id
export const reqAttrInfoList = (category1Id, category2Id, category3Id) =>
  request({ url: `/admin/product/attrInfoList/${category1Id}/${category2Id}/${category3Id}`, method: "get" });

    4、为某条spu类添加sku实例

    // 路径：/admin/product/saveSkuInfo	请求方式：post	参数：skuInfo
export const reqAddSku = (skuInfo) =>
  request({ url: `/admin/product/saveSkuInfo/`, method: "post", data: skuInfo });

    # methods

    1、initSkuData (category1Id, category2Id, row)

    1、参数：category1Id	category2Id		row 要新增sku的spu的相关数据

2、函数内容：
2.1、当在父组件Spu中，点击“添加sku实例”按钮时，调用本组件的initSkuData函数；
2.2、调用API-获取spu图片，等待服务器返回结果；如果返回结果成功，给返回的每个图片数据项添加isDefault属性并保存到data:spuImageList中；
2.3、调用API-获取spu销售属性数据，等待服务器返回结果；如果返回结果成功，将返回的数据保存到data:spuSaleAttrList中；
2.4、调用API-获取spu所属三级分类的平台属性数据，等待服务器返回结果；如果返回结果成功，将返回的数据保存到data:attrInfoList中；
2.5、将需要在表单上展示的数据，复制到data:spu中。

    2、handleSelectionChange(val) 图片选择列表中选择项发生变化时的回调函数

    1、参数：val 用户选中的行的数据组成的数组

2、函数内容：
2.1、获取用户选中行的数据，组成数组，赋值给data:selectedImgList

    3、setDefalut (row) 设置某张图片为默认图片

    1、参数：row 选择的图片所在的行数据

2、函数内容：
2.1、将图片列表数据data:spuImageList中，每一项的isDefault属性都置为false；
2.2、将带设置默认图片的isDefault属性置为true；
2.3、将data:spuImageList赋值给data:spu中的图片属性，以实现表单展示的默认图片切换。

    4、saveAdd() 保存sku实例的添加

    1、参数：无

2、函数内容：
2.1、整理参数，把用户选择的平台属性、销售属性、图片数据这三项表单数据，整理到skuInfo中。（之前是在其他数据中，现在需要向服务器提交，所以要保存到data:skuInfo中）
2.2、调用API-添加sku实例，等待服务器返回结果；如果返回成功，则弹出消息框提示，并触发Spu组件绑定的自定义事件changeScene2，向其传递消息；

    # 功能点

    1、页面数据初始化

    父组件Spu里点击“添加sku实例”按钮，调用SkuForm的页面数据初始化函数，调用相应的API，向服务器发送请求；服务器返回成功则将返回的数据保存到组件data中，完成页面数据初始化；

    2、sku属性的编辑

    用户在表单内的每步操作对应的数据变动，都对应记录在数据data中；

    平台属性：attrInfoList、 销售属性：spuSaleAttrList、图片数据：selectedImgList、其他数据：skuInfo。

    3、保存对sku属性的编辑

    用户点击保存按钮，将未保存到skuInfo中的表单数据保存过来，调用保存属性的API，向服务器发送请求；服务器返回成功则弹出消息框提示，切换回spu展示页面。

    4、取消对sku属性的编辑

    用户点击取消按钮，清空data数据，切换回spu展示页面。

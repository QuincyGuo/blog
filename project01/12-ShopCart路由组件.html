    组件介绍

    # 组件介绍

    个人购物车页面。包括去结算、修改商品数量、删除商品等。

    # 进入离开组件

    1、由Header组件，路由跳转进入；参数：无

    2、由AddCartSuccess路由组件，路由跳转进入；参数：无

    3、跳转进入Trade路由组件；参数：无

    # 生命周期

    1、mounted () 调用getData方法，调用vuex-actions-cartList：向服务器发送”获取用户购物车数据“的网络请求，如果服务器返回的result正常，则将返回的data保存到state中，以供组件使用。

    # 数据

    名称 数据类型 所属 数据介绍
    cartList array computed 组件仓库state中获取到的服务器返回的购物车相关数据
    cartInfoList array computed 从cartList中分解出来的购物车数据
    totalPrice number computed 购物车内商品总价
    isAllCheck boolean computed 购物车内所有商品是否都被选中

    # API

    1、改变购物车商品数量

    地址：/api/cart/addToCart/{skuId}/ {skuNum}  请求方式：post    需要带参数skuId, skuNum
export const reqAddOrUpdateCart = (skuId, skuNum) =>
  requests({ url: `/cart/addToCart/${skuId}/${skuNum}`, method: "post" });
// skuNum的含义是：对购物车内已有数量的增减数量，正数代表对目前数量增加多少，负数代表减少多少

    2、获取购物车相关数据

    地址：/api/cart/cartList  请求方式：get    参数：无
export const reqGetCartList = () => 
  requests({ url: "/cart/cartList ", method: "get" });

    3、删除购物车内商品

    地址：/api/cart/deleteCart/{skuId}   请求方式：DELETE	需要带参数skuId
export const reqDeleteCart = (skuId) =>
  requests({ url: `/cart/deleteCart/${skuId}`, method: "DELETE" });

    4、切换商品选中状态

    地址：/api/cart/checkCart/{skuId}/{isChecked}   请求方式:get   需要带参数skuId，isChecked(string类型的1/0)
export const reqUpdateCartChecked = (skuId, isChecked) =>
  requests({ url: `/cart/checkCart/${skuId}/${isChecked}`, method: "get" });

    # vuex

    vuex-shopcart：
1、actions
1.1、getCartList
调用API-获取购物车相关数据；服务器返回请求成功，则调用mutations-GETCARTLIST
1.2、deleteCart
调用API-删除购物车内商品；服务器返回请求成功，则通知组件成功
1.3、deleteAllCheckedCart  
通过getters获得所有状态为选中的商品id，然后对每一个都调用deleteCart的actions，然后将所有的请求结果合并进行判断，将最终结果返回给组件

2、mutations
2.1、GETCARTLIST
将服务器返回的商品详情数据保存到state的cartList中，以供其他组件调用

3、state
cartList:[] 			// 购物车相关数据，数组

4、getters
cartList				// 购物车相关数据，简化state数据读取，对象

    # 功能点

    1、页面数据动态展示

    组件在mounted阶段，调用getData方法；vuex调用获取购物车相关数据的API，向服务器发送请求；服务器返回成功则将返回的data保存到state中；本组件的computed获取vuex-state数据，并保存为计算属性；页面在展示时，使用相应的属性即可。

    2、修改购物车内商品数量

    用户点击页面按钮，组件计算修改数量的变化值，调用vuex的addOrUpdateCart；vuex调用修改购物车内商品数量的API，向服务器发送请求；服务器返回成功，组件调用getData方法；后面与1一致，向服务器获取修改后的个人购物车数据的请求，以刷新页面。

    3、删除某一商品

    用户点击删除按钮，组件调用vuex的deleteCart；vuex调用删除购物车内商品的API，向服务器发送请求；服务器返回成功，组件调用getData方法；后面与1一致，向服务器获取修改后的个人购物车数据的请求，以刷新页面。

    4、删除所有选中的商品 用户点击按钮，组件调用vuex的deleteAllCheckedCart；（该函数通过getters获得所有状态为选中的商品id，然后对每一个都调用deleteCart的actions，每一个都向服务器发送一次请求），然后将所有的请求结果合并进行判断，将最终结果返回给组件；成功，则组件调用getData方法；后面与1一致，向服务器获取修改后的个人购物车数据的请求，以刷新页面。

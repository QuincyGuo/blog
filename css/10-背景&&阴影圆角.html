    背景属性

    # 背景属性

    # 默认值

    background-color: 		transparent（透明）
background-image: 		none（无）
background-repeat: 		repeat（重复）
background-attachment: 	scroll（滚动）
background-position: 	0 0（等价于top left，即左上角）

默认情况下（使用标准盒模型时），元素的背景会填充其内容和内边距所在的区域。如果设置了position，超出元素边框部分的就不显示。

    # background-repeat：背景图片是否平铺

    repeat（同时横向纵向平铺）
no-repeat（不要平铺）
repeat-x（横向平铺）
repeat-y（纵向平铺）

比如，用一张宽度为1px，颜色纵向渐变的图片，然后让他横向平铺，整个页面就都是渐变的了。

    # background-position：背景图片定位

    background-position:向右偏移量 向下偏移量（背景图片的偏移）;

属性值可以是正数，也可以是负数（距离四条边的距离。正数向内偏移，负数向外偏移。默认是以左边、上边为标线）。
比如：100px 200px（以左边为线向内偏移，以上边为线向下偏移）、-50px -120px（以左边为线向外偏移，以上边为线向上偏移）。

    background-position: 描述左右的词 描述上下的词;

描述左右的词：left、center、right
描述上下的词：top 、center、bottom
这里的位置词语，表示背景图片和元素所占实际大小之间（包括margin）的对齐方式。

    # background-attachment：背景图片是否随页面滚动

    scroll:默认值，随页面滚动；
fixed:页面滚动，背景图片不动。

    # background-size：背景图片尺寸

    500px 500px;
50% 50%;   		//* 宽高的百分比（相对于元素的大小） */
cover; 	  		//* 图片以固定长宽比放大，以始终填充满容器；超出部分被隐藏。 */
contain;		//* 图片以固定长宽比放大，以保证高或宽中较大的与容器持平。可能会导致容器的部分区域为空白。 */

    # linear-gradient线向渐变 radial-gradien 径向渐变

    background: linear-gradient(to bottom right, red, blue);
background: linear-gradient(290deg, red, blue);
background: linear-gradient (#f90 0, #f90 2%, #555 28, #eee 50%, #555 98%, #f90 98%, #f90 100%);
//数值代表的是圆周上的点的位置，渐变结束的位置：0代表最顶端的点，90代表最左边的点，180代表最底端的点，270代表最右边的点。
//因为是结束位置，所以是逆时针。反正用的时候自己试就行了。

    background: radial-gradient(at top, yellow, red);	
//这个at是渐变开始的位置。
background: radial-gradient(100px 50px, yellow, red);
//这两个数值是渐变尺寸。水平、竖直方向各渐变多少距离。

background: 
radial-gradient(100px at center, 
    yellow 0%,
    green 30%,
    blue 60%,
    red 100%);
}

/* 手动制作渐变效果是比较困难的。可以使用线上的渐变效果生成器。 */

    # 阴影

    水平和垂直偏移：向右向下为正，向左向上为负。

    # box-shadow 元素边框阴影

    box-shadow: 水平偏移 垂直偏移 模糊半径 阴影扩展半径 阴影颜色，其中模糊程度和阴影大小是可选的。

    box-shadow: 0 10px 20px rgba(0,0,0,0.19), 0 6px 6px rgba(0,0,0,0.23);
<!-- 可以通过逗号分隔给元素添加多个阴影 
	设置边框阴影不会改变盒子的大小，即不会影响其兄弟元素的布局。只是视觉效果。
	阴影扩展半径为负数，表示阴影不会向外扩张，而是向内收缩。（阴影四边的宽高比元素窄）
	内阴影：在阴影颜色后面加上inset关键字。
-->

    <!-- 用box-shadow属性设置出一个月亮图形：给圆形设置圆形的阴影，两个圆形交叉减去的部分就是月形。-->

    width: 100px;
    height: 100px;
    background-color: transparent;
    border-radius: 50%;
    box-shadow: 25px 10px 0px 10px blue;
  }
</style>

    # text-shadow 文本阴影

    text-shadow属性接受四个值：水平偏移 垂直偏移 模糊半径 阴影颜色。（向右向下为正，向左向上为负）
水平偏移，垂直偏移，可以是负数、正数、0；
模糊半径，不能是负数，如果不写，默认是0。

text-shadow: 1px 1px 1px white,-1px -1px 1px black;
比如上面这句代码，给左上角放白色的阴影，右下角放黑色的阴影，就达到了文字凹下去的效果。

    # border-radius 设置圆角的原理

    border-radius属性，能给盒子/元素设置圆角，其原理是：

    用属性设置的圆/椭圆，与元素边框重合，切掉角落里不重合的部分，则元素产生了圆角。

    /* 单独一个角设置 */
border-top-left-radius：30px;		/* 圆角 */	/*半径为30px的圆，与边框的交集，形成圆角。*/
border-top-left-radius：30px/50px;	/* 椭圆角，/以左为水平半径，以右为竖直半径 */	/*水平半径30px，竖直半径50px的椭圆。*/
border-top-left-radius：30%/50%;		/* 百分比圆角、椭圆角 */		/*水平半径为元素宽30%，竖直半径为元素高50%，的椭圆。*/


/* 四个角设置 */
border-radius：30px;					/* 四个角均为半径30px的圆角 */
border-radius：30px 40px 50px 60px;	/* 四个角从左上角顺时针依次为半径30px 40 50 60px的圆角 */
border-radius：30px / 50px;			/* 四个角均为半径30px/50px的椭圆角 */

border-radius：30% 40% 50% / 20px 40px;	
/* 先看反斜杠，左边为水平半径，右边为竖直半径；
再看四个角的顺序，四个角水平半径依次为元素宽的30%，40%，50%，40%，四个角竖直半径依次为20px，40px，20px，40px*/

    重点：

    先看，有反斜杠的，左为水平半径，右为竖直半径；

    再看，元素四个角的四边顺序；

    最后看，有百分号的，代表椭圆的半径是元素的宽、高的对应百分比。

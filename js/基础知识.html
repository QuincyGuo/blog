    JavaScript介绍

    # JavaScript介绍

    # 前端、解释型语言

    前端是指运行在用户终端（浏览器）上，而不是服务器上；不能操作数据库。
后台语言运行在服务器上，比如PHP，能操作数据库。
node.js是用JavaScript开发的后台语言。
	
解释型：不需要提前整个编译成机器语言，而是翻译一行执行一行。

    # JavaScript的组成

    ECMAScript：			JavaScript是语法标准，包括变量、语句、表达式、函数等语法标准。
DOM（文档对象模型）：	操作HTML页面上元素的API。
BOM（浏览器对象模型）：	操作浏览器部分功能的API。

    # 变量

    # 声明

    ES6语法之前：var name;
ES6语法之后：const name;常量	let age;变量

    # 命名

    变量名区分大小写。最佳的命名方法是驼峰法，比如let myFirstName;
常量的常见命名做法是全部使用大写字母，单词之间用下划线分隔，比如const BLACK_COLOR;

    # 基本数据类型

    String 字符串
Number 数值
Boolean 布尔值
Null 空值
Undefined 未定义

    # 引用数据类型

    object 对象
内置对象Function、Array、Date等都是Object类型。除了上面的五种基本类型，其余的都是Object类型。

    # 栈内存与堆内存

    基本数据类型
JS中，所有的变量都是保存在栈内存中的。值与值之间是独立存在，修改一个变量不会影响其他的变量。

引用数据类型
对象内容是保存到堆内存中的。每创建一个新的对象，就会在堆内存中开辟出一个新的空间；
而对象名（变量）保存的是对象的内存地址（对象的引用），保存在栈内存当中。

    # 引用数据类型，对象内容与变量的不同保存的问题

    var obj1 = new Object();
obj1.name = 'smyh';
var obj2 = obj1;
obj1.name = 'vae';
console.log(obj1.name); // 打印结果：vae
console.log(obj2.name); // 打印结果：vae
变量obj1 和 obj2 都是引用数据类型，让 obj2 等于 obj1，两个变量保存的都是对象的地址，然后修改 obj1.name 的值之后，实际修改的是对应地址的对象，所以 obj2.name 的值也发生了改变。
所以，这句var obj1 = new Object();实际上是分为两部分。先是new Object()，新创建一个对象，然后把内容地址赋值给obj1。

    如何不影响地把对象复制？
var obj1 = new Object();
obj1.name = 'smyh';
var obj3 = Object.assign({}, obj1); 	//通过 Object.assign() 来复制对象
这样，修改obj1就不会影响obj3的内容，因为obj3有单独的对象内容保存在堆内存中。

    # switch case

    switch的原理：遇到符合的case就进入，然后结束的条件是：遇到 break 就结束或者执行到程序的末尾就结束。

switch 语句的结束与否和 default 的顺序无关，但是建议放到最后。

实际例子：
var number = 5;
switch (number) {
    default:
        console.log('我是defaul语句');
    case 2:
        console.log('第二个呵呵:' + number);
    case 3:
        console.log('第三个呵呵:' + number);
        break;
    case 4:
        console.log('第四个呵呵:' + number);
        break;
}
这段代码的输出结果是：
我是defaul语句
第二个呵呵:5
第三个呵呵:5
//理由是default和case2都没有break，所以发生case穿透，没有结束，即使后面的case2不匹配，也会继续执行后面的语句直到break。

    # 从其他js文件导入函数

    import * as myMathModule from "./math_functions.js";
上面的 import 语句会创建一个叫作 myMathModule 的对象。 这只是一个变量名，可以随便命名。 
对象包含 math_functions.js 文件里的所有导出，可以像访问对象的属性那样访问里面的函数。 
下面是使用导入的 add 和 subtract 函数的例子：

import { add, subtract } from './math_functions.js';
// 导入一部分函数

myMathModule.add(2,3);
myMathModule.subtract(5,3);

    对象简介

    # 对象简介

    对象是一组无序的属性和方法的集合。对象的作用是封装信息。

    保存一个值，用变量；保存一组类型一致的值，用数组。

但是如果要保存一组具有相关性但是数据类型又五花八门的数据时，用数组就不合适。比如，var arr = ['王二', 35, '男', '180'];
这时，就需要用对象保存：
var person = {};
person.name = '王二';
person.age = 35;
person.sex = '男';

    # 对象的内存保存/传值传址

    对象的值是保存在堆内存中的，而对象的引用（即变量）是保存在栈内存中的。如果两个变量保存的是同一个对象引用，当一个通过一个变量修改属性时，另一个也会受到影响。

实际例子：
var obj1 = new Object();
var obj2 = obj1;		//obj1 和 obj2 指向了同一个堆内存空间
obj1.name = 'vae';		//结果：obj2.name 的值也发生了改变

如何不影响地把对象复制？
var obj3 = Object.assign({}, obj1); 	//通过 Object.assign() 来复制对象

    # Object.assign复制

    let obj = { name: "", sex: "", age: "" };
let obj1 = { name: "bob", sex: "male" };
let obj2 = { name: "", age: "22" };
Object.assign(obj, obj1, obj2);
console.log(obj);		// 结果是：{name: "", sex: "male", age: "22"}
// 也就是，会把同名的属性复制给第一个对象，但是后面的会覆盖掉前面的。

    # 对象的分类

    # 内置对象

    由ES标准中定义的对象，比如：Object、Math、Date、String、Array、Number、Boolean、Function等。（这里的String和Number是对象名，不是数据类型string，number）

    # 宿主对象

    由JS的运行环境提供的对象，目前来讲主要指由浏览器提供的对象。

    比如 BOM DOM。比如console、document。

    # 自定义对象

    由开发人员自己创建的对象，通过 new 关键字创建出来的对象实例。

    # 对象的创建

    # 方式一：var obj = {};

    也就是创建对象名时就把内容也建好。

    const obj2 = {
    name: "千古壹号",
    age: 26,
    test: {							// 还可以存放一个嵌套的对象
        id: 123,
        tel: 180
    }    
    sayName: function() {			//增加一个方法。以后可以通过obj2.sayName()的方式调用这个方法
        console.log(this.name);
    }
};

    # 方式二：new Object()

    本质上是先 new Object()创建对象名，后面再添加内容。但是创建出的对象都是Object这种类型，无法自定义。

    function createPerson(name, age, gender) {   
    var obj = new Object();			//创建一个新的对象
    obj.name = name;				//向对象中添加属性
    obj.age = age;
    obj.gender = gender;
    obj.sayName = function () {
        alert(this.name);
    };    
    return obj;						//将新的对象返回
}

var obj2 = createPerson('猪八戒', 28, '男');
var obj3 = createPerson('白骨精', 16, '女');

    # 方式三：构造函数

    function Student(name) //自定义Student对象的构造函数
{
	this.name = name; //this指的是当前对象实例
	this.sayHi = function () {
    	console.log(this.name + '厉害了');
	};
}
var stu1 = new Student('smyh');		//利用构造函数自定义Student对象，如果没有前面的Student构造函数，Student关键字就是无意义的

    # 与普通函数的区别

    - 构造函数习惯上首字母大写
- 普通函数是直接调用，而构造函数需要使用 new 关键字来调用。比如new Student('smyh');
- this的指向也不同：
	1.以函数的形式调用时，this 永远都是 window。比如fun();相当于window.fun();
	2.以方法的形式调用时，this 是调用方法的那个对象
	3.以构造函数的形式调用时，this 是新创建的实例对象

    # new一个构造函数的流程

    1. 开辟内存空间，在内存中创建一个新的空对象（也就是对象实例）
2. 让this指向这个新的空对象
3. 执行构造函数的代码，给新对象添加属性和方法
4. 返回这个新对象（所以构造函数里面不需要 return）。但是new调用时的返回值，显式返回函数或者对象，比如下面，返回的就不是新生成的对象。
	// return function f(){};
    // return {};

    # 静态成员和实例成员

    JavaScript 的构造函数中可以添加一些属性、方法。
静态成员：在构造函数本身上添加的成员称为静态成员，只能由构造函数本身来访问。
实例成员：在构造函数内部的this上添加称为实例成员，只能由实例化的对象来访问。

实际例子：
function Person(name,age) {
	this.name = name;				//这里的name/age属性，sayHi方法都是实际成员
	this.age = age;
	this.sayHi = function(){
		console.log('Hello...');
	} 
	Person.hobby = 'Irunning';
  	Person.climb = function(){
    	console.log('Climbing...')
    } 
}
var p1 = new Person('Lucy',29);

结果：
p1.name;
p1.sayHi();
Person.climb();
Person.hobby;		//以上是正确的，静态成员只能由构造函数本身来访问；实例成员只能由实例化的对象来访问
p1.climb();			//报错
Person.sayHi();		//报错

    # 类&实例

    将一个构造函数称为一个类。
使用同一个构造函数创建的对象，我们称为一类对象，也将这些对象，称为该类的实例。
Object类是所有类的祖先。因为Student类是这么来的，var Student = new Object();

    # 类的创建

    class Rectangle {
    // constructor 构造函数
    constructor(height, width) {
        this.height = height;
        this.width = width;
    }
    // Getter 外传属性
    get area() {
        return this.calcArea()
    }
    // setter 传进并修改属性
    set writer(updatedAuthor) {
    	this._width = updatedAuthor;
  	}
    // Method 普通方法
    calcArea() {
        return this.height * this.width;
    }
    // static 静态方法。
    static distance(a, b) {
        const dx = a.x - b.x;
        const dy = a.y - b.y;
        return Math.hypot(dx, dy);
        // 通过类名调用静态方法，不能通过一个类实例调用静态方法。
        // p1.distance;错误		Rectangle.distance(p1, p2);正确
    }
}

    # 对象（实例）的基本操作

    # 获取对象属性

    特殊属性名：
比如说，123、my这种非字符串或有空格的属性名，用 对象['属性名'] = 属性值;
obj.123 = 789		//错误
obj['123'] = 789;	//正确


属性名是在一个变量里：
let zhuName = 'name';	//变量zhuName的内容必须要和对象obj的属性名一致，也就是obj要有一个属性名为name的属性。
obj[zhuName] = 789;		//变量名不加引号


如果获取对象中没有定义的属性，不会报错而是返回undefined。

    # 检查对象是否有该属性

    我们平时使用的对象不一定是自己创建的，可能是从接口获取的，这个时候，in 运算符可以派上用场。

console.log('name' in obj);			//'属性名' in 对象;			
if (obj.name) {						//另一种写法：如果对象obj中有name属性，我就。。。
}

    # for in 对象循环

    对象（实例）中有几个属性，循环体就会执行几次。每次执行时，会将对象中的每个属性的 属性名 赋值给变量。
for (var key in obj) 					//key不是关键字，而是可变的变量名
{
    console.log('属性名:' + key); 		// 这里的 key 是：对象属性的键（也就是属性名）
    console.log('属性值:' + obj[key]); // 这里的 obj[key] 是：对象属性的值（也就是属性值）
}

嵌套对象的for in遍历：
const usersObj = {		//嵌套对象
  Alan: {
    online: false
  },
  Jeff: {
    online: true
  }
}
//使用usersObj[user]循环第一层所有对象
for (let user in usersObj) {
    if (usersObj[user].online === true) {
      。。。
    }
}

    # Object.defineProperty方法

    会直接在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回此对象。
const object1 = {};
Object.defineProperty(object1, 'property1', {
  value: 42,
  writable: false,		// 该属性能否被重新赋值。
  enumerable: true, 	// 该属性是否可以在 for...in 循环和 Object.keys() 中被枚举。
  configurable: false 	// 该属性是否可以被删除
  
  // 当property1属性被读取时，get函数就会被调用，返回值就是property1属性值
  get: function() {
      console.log('属性被读取，get被调用');
      return property1;
  },
  
  // 当property1属性被修改时，set函数就会被调用，且会收到修改的值
  set: function(value) {
  	  console.log('属性被修改，set被调用');
      property1 = value;
  }
});

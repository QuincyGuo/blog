    组件介绍

    # 组件介绍

    商城首页。包括全部商品分类、主要banner轮播、商品分类介绍、底部商城信息等。

    # 进入离开组件

    1、当用户刚由外部访问进入网站项目时，重定向路由跳转至到Home组件；

    2、由Header静态组件，路由跳转进入，无参数。

    # 子组件

    TypeNav三级导航分类静态组件，ListContainer静态组件，Floor静态组件

    # 生命周期

    1、mounted () 调用vuex-actions-getFloorList：向服务器发送”获取商品楼层floor数据“的网络请求，如果服务器返回的result正常，则将返回的data保存到state中，以供组件使用。

    # 数据

    名称 数据类型 所属 数据介绍
    floorList 数组 computed 组件从vuex中获取到的服务器返回的floor数据

    # API

    1、获取floor数据

    地址：/api/floor   请求方式：get    无参数
export const reqGetFloorList = () =>
  mockRequests({ url: "/floor", method: "get" });

    # vuex

    vuex-home：
1、actions
1.1、getBannerList
调用API-获取轮播图banner数据；服务器返回请求成功，则调用mutations-BANNERLIST
1.2、getFloorList
调用API-获取商品楼层floor数据；服务器返回请求成功，则调用mutations-FLOORLIST
 
2、mutations
2.1、BANNERLIST
将服务器返回的轮播图banner数据保存到state的bannerList中，以供其他组件调用
2.1、FLOORLIST
将服务器返回的商品楼层floor数据保存到state的floorList中，以供其他组件调用

3、state
bannerList: [], 		// 轮播图banner数据，数组
floorList: [], 			// 商品楼层floor数据，数组

    # 功能点

    1、页面数据动态展示

    组件在mounted阶段，调用vuex的getFloorList；vuex调用获取商品楼层floor数据的API，向服务器发送请求；服务器返回成功则将返回的data保存到state中；本组件的computed获取vuex-state数据，并保存为计算属性；并将数据传递给Floor子组件。

    # 轮播图 Swiper插件

    Swiper是一个实现轮播图的js插件，将其定义为全局组件。

    # 组件定义

    1、下载插件；
2、components文件夹下，创建Carousel文件夹，里面创建index.vue文件；
3、index.vue文件中引入Swiper包；
import Swiper from 'swiper';  // 引入Swiper的js包
// 还需在main.js里引入Swiper插件样式

4、添加html
<template>
  <div class="swiper-container"
       id="floor1Swiper">
    <div class="swiper-wrapper">
      <div class="swiper-slide"
           v-for="(carousel) in list"
           :key="carousel.id">
        <img :src="carousel.imgUrl" />
      </div>
    </div>
    <!-- 如果需要分页器 -->
    <div class="swiper-pagination"></div>

    <!-- 如果需要导航按钮 -->
    <div class="swiper-button-prev"></div>
    <div class="swiper-button-next"></div>
  </div>
</template>

5、初始化Swiper实例
watch: {  // 此处也可以写在mounted里，因为组件本身没有发送请求数据行为，数据是由父组件传递过来的，和组件本身的DOM更新没有关系
    immediate: true,  // 此处要加这句是，这个数据是父组件传过来的，后面再没有变化过，所以watch监听不到变化，里面的语句没执行，所以轮播图不起作用。加上这句表示，数据立即监听，不论数据有无发生变化。
    bannerList: {   
      handler () {
        this.$nextTick(function () {
          new Swiper('.swiper-container', {
            loop: true, // 循环模式选项
            // 如果需要分页器
            pagination: {
              el: '.swiper-pagination',
              clickable: true,
            },

            // 如果需要前进后退按钮
            navigation: {
              nextEl: '.swiper-button-next',
              prevEl: '.swiper-button-prev',
            },
          })
        })
      }
    }
  }
